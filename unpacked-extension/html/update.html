ImageQuilts v3

Welcome to ImageQuilts Version 3. With this exciting new release, you can do more than ever with ImageQuilts.

New features

  1. Export — You can now download your quilts using the new "Download Quilt" button.
  2. Drag-and-drop — add images from your computer to an existing quilt.
  3. Create quilts from any webpage — visit any page and click the extension icon to create a quilt.
  4. Right-click to add — add individual images from any page to add to a quilt.
  5. More flexible scaling and zooming.
  6. Performance enhancements, bug fixes, and more!

Thanks for all your feedback and support. Happy quilting.
